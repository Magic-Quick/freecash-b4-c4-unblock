# ASSET_SPEC — «Unblock» (Freecash B4C4)

> Спека placeholder-ассетов для `cocos-asset-maker`. Все — временные болванки; дизайнеры заменят
> после нарезки финальных ассетов (см. OPEN_ISSUES #1).
> **Арт-направление (утв.): `concept-1.png` — классическое дерево (Unblock Me).** Деревянные блоки,
> тёплый уютный фон. Главный блок — красный, выезжает **как блок (без машинки)**.

## Общий стиль
- Мультяшный, тёплые насыщенные цвета, чёткие скруглённые формы, мягкие тени (плоские, без drop-shadow).
- Скругления: 10–20% от меньшей стороны. Контур: тёмная обводка 2–3px alpha 0.6.
- Блики: белый overlay 20–30% сверху на выпуклых объектах. Anti-alias вкл., без blur.
- Текст на ассетах («FC», цифры) рисовать **формой**, не шрифтом.
- **Без денежных символов** ($, купюры, банк-карты) — требование Freecash (GDD §2). Монета — только «FC».

## Правила качества
- PNG RGBA8 без потерь; прозрачность только где нужно (`background` — без alpha).
- 9-slice-ассеты (`block_tile`, `block_main`, `board_frame`, `panel`, `button_play`) рисовать так, чтобы
  углы не растягивались — это позволяет любому размеру блока/панели без новых файлов (минимум ассетов).
- Никаких `.meta` вручную — Cocos создаст при импорте.
- Держать суммарный вес в рамках бюджета 5 MB (GDD §10) — placeholder'ы небольшие.

## Цвета (концепт-1, классическое дерево)
Препятствия — `wood-orange #E0982E` (единый). Главный блок — `red #E24A3B`. Ячейка — `#6E5138`,
рамка — `#C9A063`. Тинт `block_tile` через `Sprite.color` оставлен как опция, но по умолчанию все препятствия одинаковые.
