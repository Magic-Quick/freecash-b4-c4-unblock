# QA_CHECKLIST — «Unblock» (Freecash B4C4)

> Финальный ручной прогон + статические проверки. Часть гейта Фазы 7.

## Smoke-сценарий (core loop)
1. Запуск сцены → INTRO: поле L1 видно, главный (красный) блок выделен, стрелка выхода справа.
2. Палец-подсказка пульсирует/свайпает первый мешающий блок; после первого хода палец скрывается.
3. Свайп мешающих блоков вдоль их оси — блоки скользят до упора, упор в стенку/блок → «клак», сдвиг = 0 не двигает.
4. Коридор главного блока расчищен → главный (красный) блок автоматически едет к выходу и выезжает за правый край (без машинки, концепт-1).
5. Монеты FC летят к счётчику, счётчик = **9**.
6. Авто-переход на L2 (больше блоков). Расчистка пути → выезд → счётчик = **19** (итого).
7. Появляется CTA «PLAY & EARN» (панель: лого Freecash, «LEVEL COMPLETE», 19 FC).
8. Клик по кнопке → `plbx.download()` (в билде — редирект на стор).

## Функциональные проверки
- [ ] Горизонтальные блоки двигаются только влево/вправо; вертикальные — только вверх/вниз.
- [ ] Блок не проходит сквозь другой блок и сквозь стенку.
- [ ] Главный блок едет только когда коридор реально свободен (иначе упирается — `EVT_MAIN_BLOCKED`).
- [ ] Оба уровня решаемы; раскладки берутся из `assets/data/levels.json`.
- [ ] Счётчик FC: 0 → 9 → 19; без знака $, только «FC».
- [ ] CTA появляется ТОЛЬКО после решения L2 (не раньше — network-safe).
- [ ] Дисклеймер «For illustration purposes only» виден на протяжении всего плейбла.
